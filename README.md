# Instruções

## Pré-requisitos
```bash
    ### Instalar Node e npm
    sudo apt install npm -y
    sudo apt install nodejs -y 
    ### intalar dependencias para o javascript
    npm install sqlite3
    npm install sqlite
    npm install express
    ### Compilar no diretorio minha revenda
    node app.js
```
## Visualizar na porta
http://localhost:3000/
